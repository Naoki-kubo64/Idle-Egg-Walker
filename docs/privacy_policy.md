# プライバシーポリシー (Privacy Policy)

**最終更新日: 2025 年 12 月 24 日**

本プライバシーポリシーは、「Egg Walker」（以下「本アプリ」）における利用者情報の取り扱いについて定めたものです。

## 1. 取得する情報とその利用目的

本アプリは、サービスの提供と向上のために、以下の情報を取得・利用します。

### 1.1 ヘルスケアデータ（Health Connect）

本アプリは、Google Health Connect API を使用してお客様の歩数データを取得します。

- **取得するデータ:** 歩数 (Steps)
- **利用目的:** ゲーム内の経験値（EXP）への変換、および歩数履歴グラフの表示のため。
- **データの共有:** 取得したヘルスケアデータが、広告表示やその他の目的で第三者に共有されることはありません。また、本アプリのサーバーに送信・保存されることもありません（データは端末内でのみ処理されます）。

### 1.2 広告配信（AdMob）

本アプリでは、広告配信のために Google AdMob を使用しています。
AdMob は、適切な広告を表示するために、Cookie やデバイス ID などの情報を自動的に取得する場合があります。
Google のプライバシーポリシーや広告設定については、以下のリンクをご参照ください。

- [Google ポリシーと規約](https://policies.google.com/technologies/ads)

### 1.3 アプリ内課金（RevenueCat）

本アプリでは、アプリ内課金機能を提供するために RevenueCat を使用しています。
課金処理およびサブスクリプション管理のために、匿名化されたユーザー ID や購入履歴情報が RevenueCat のサーバーに送信される場合があります。

- [RevenueCat Privacy Policy](https://www.revenuecat.com/privacy)

## 2. 情報の第三者への開示

本アプリは、以下の場合を除き、取得した個人情報を第三者に開示することはありません。

- 法令に基づく場合
- ユーザーの同意がある場合
- 利用目的の達成に必要な範囲で、外部サービス（Google AdMob, RevenueCat など）に委託する場合

## 3. データの保存と削除

本アプリのゲームデータ（セーブデータ）は、主にお客様の端末内に保存されます。アプリをアンインストールすることで、これらのデータは削除されます。

## 4. プライバシーポリシーの変更

開発者は、必要に応じて本プライバシーポリシーを変更することがあります。重要な変更がある場合は、アプリ内または本ページにて通知します。
